План видов расчета предназначен для описания структуры хранения информации о возможных видах расчетов.  По плану видов расчетов система создает в базе таблицу, где хранится информация о видах расчета и взаимосвязях между ними.

План видов расчета обладает  свойствами:

 "Использует период действия", что позволит системе вытеснять каждый вид расчета по периоду действия.

План видов расчета  зависит от базы, т.е. будут ли в этом плане виды расчетов, зависящие от базы.  Зависимость от базы может быть по периоду действия или по периоду регистрации.

 

 

 

 

Регистр расчета  предназначен для описания структуры накопления данных в результате выполненных расчетов. Основное предназначение регистра расчета - это занесение в него информации по заложенному алгоритму. Для интерактивного редактирования регистра расчета не предназначен.

Отличительной возможностью регистра расчетов является его периодичность, благодаря которой можно использовать механизм вытеснения по периоду действия и зависимости от базового периода, связанного с планом видов расчета.
