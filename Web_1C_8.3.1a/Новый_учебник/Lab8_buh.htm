Лабораторная работа №11. Задачи бухгалтерского учета

Бухгалтерская запись называется проводкой.  Для  формирования проводки требуется специализированный справочник, План счетов. Проводка, сформированная только с использованием плана счетов содержит верную информацию, но не очень понятную. Такая проводка называется синтетической, а денежные операции с использованием таких проводок называют синтетическим учетом.

Чтобы получить проводку, содержащую пояснения выполненной операции, используют специальный объект План видов характеристик и справочник Субконто тогда полученная проводка называется аналитической, а денежные операции с использованием таких проводок аналитическим учетом, рисунок 1а.

Техническое задание

Для решения задач бухгалтерского учета потребуется создать несколько объектов:

  1. план счетов,
  2. регистр бухгалтерии,
  3. план видов характеристик,
  4. справочник "Субконто",
  5. документ, который будет формировать проводки.

Для ведения задач по бухгалтерскому учету требуется в состав конфигурации ввести несколько дополнительных объектов, специфичных для бухгалтерии:

  • план счетов;
  • справочник "Субконто";
  • план видов характеристик;
  • регистр бухгалтерии.

План счетов - это специализированный справочник, используемый при автоматизации бухгалтерского учета. Записи справочника представляют собой описание бухгалтерских счетов, которые имеют помимо наименования специальный код. Бухгалтерские счета используются для формирования бухгалтерской записи (проводки), которая свидетельствует о проведенной денежной операции. Сформированная проводка сохраняется в регистре бухгалтерии. Бухгалтерская запись без расшифровки называется синтетическим учетом, расшифрованная бухгалтерская запись называется аналитическим учетом, структура бухгалтерских записей обоих типов приведена в соответствии с рисунком 1.
