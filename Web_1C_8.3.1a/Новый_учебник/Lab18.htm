Дополнение к ТЗ на разработку документа «Кассовые Выплаты»

Требуется самостоятельно разработать форму документа «Кассовые выплаты». Документ предназначен для кассира, он оформляется при выдаче денег подотчетному лицу. Подотчетное лицо – это сотрудник, кому выплачивают деньги на некоторые виды расходов. Форма документа «Кассовые выплаты» используется для формирования унифицированной формы «Расходный кассовый ордер». Унифицированная форма – это стандартная форма документа, утвержденная Государственным комитетом по статистике и используемая на территории РФ. Унифицированная форма КО-2 к заданию прилагается.

Макет формы будет строиться на основании готовой формы, созданной  в среде Excel, и имеющей расширение xls, рисунок Д-1.

Рисунок Д-1 – Форма Расходного кассового ордера

Графа «Выдать» заполняется из справочника СОТРУДНИКИ

графа «Структурное подразделение» также заполняется из справочника СОТРУДНИКИ (реквизит ОТДЕЛ)

Графа «основание» заполняется в зависимости от значения переключателя ОСНОВАНИЕ. Его значения могут быть следующими: командировка, код 01; приобретение ТМЦ, код 02; мат.помощь, код 03; прочее, код 04. Значение реквизита «код» проставляется в ячейку КОД ЦЕЛЕВОГО НАЗНАЧЕНИЯ. Значение СУММА проставляется цифрами и прописью. (ТМЦ – товары малоценные).

Графа «Приложение» заполняется в зависимости от значения графы «Основание»:

-                 Если основание – материальная помощь, то основанием может быть заявление

-                 Если основание – командировка, то основание приказ №__ дата__

-                 Если основание – приобретение ТМЦ, то основание требование №__ дата __

-                 Если основание – прочее, то в графе пишется Прочие расходы

В графе «По» записываются паспортные данные получателя денег.

Графа «Дебет. Корреспондирующий счет» заполняется значением 71 из плана счетов. Счет 71 – сотрудники

Графа «Кредит» заполняется значением 50 из плана счетов. Счет 50 - касса

Номер документа и дата составления берется из соответствующих реквизитов формы.

Предусмотреть, чтобы при выборе того или иного значения реквизита ОСНОВАНИЕ открывалась видимость необходимой информации:

-                 Если основание – приобретение ТМЦ, то должны открываться поля для занесения  № требования и даты

-                 Если основание – командировка, то должны открываться поля для введения номера приказа и даты

-                 Если основание – материальная помощь, то дополнительные поля не открываются, а в поле приложение заносится «заявление»

-                 Если основание – прочее, то дополнительные поля не открываются, а поле приложение  заносится «Прочие расходы»

Для заполнения графы «По» необходимо, чтобы в поле формы были ячейки для заполнения номера и серии паспорта, даты выдачи и указания кто выдал. Для заполнения номера и серии предусмотреть маску, если это необходимо.

Предложенная для примера процедура формирует документ и записывает его по указанному адресу. Затем открывает его для просмотра. Для открытия созданного документа используется СОМ объект.

При написании строки кода Excel=Новый COMОбъект("Excel.Application") СОМ пишется латиницей.

Ознакомиться с объектами СОМ можно в электронном учебнике на главной странице, ссылка Передача объектов 1С:Предприятия через COM
